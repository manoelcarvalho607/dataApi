# Objetivo do projeto

- API REST com java e Spring Boot 
- Banco de dados com ORM
- Front end com React Js
- Camadas, componentes
- Integração com SMS 
- Implantação na nuvem com CI/CD

## tecnologias usadas

- spring (spring boot, Web,JPA,H2,Security)
- react js
- javaScript
- java
- Datepicker
- Twilio
- axios
- toastify

## deploy
- Heroku (backend)
- netLify (frontend)
